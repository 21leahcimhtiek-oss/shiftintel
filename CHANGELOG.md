# Changelog

All notable changes to ShiftIntel are documented here.

## [1.0.0] - 2024-02-01

### Added
- Initial production release
- AI schedule generation using GPT-4o
- Weekly calendar view with drag-to-publish shifts
- Employee roster with skills and availability management
- Time-off request workflow (request → approve/deny)
- Coverage rules engine (min staffing per department/time slot)
- Labor cost forecasting (4-week projections)
- Real-time coverage score (0-100%)
- Stripe billing with Starter/Pro/Enterprise plans
- Row Level Security for complete org data isolation
- Rate limiting on all API routes
- Sentry error monitoring
- E2E tests with Playwright
- Unit tests for core business logic
